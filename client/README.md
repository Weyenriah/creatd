# Frontend - Creatd
This is the frontend for Creatd App.

## Project Setup

```sh
cd client/
npm install
```

### Compile and Hot-Reload for Development

```sh
cd client/
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
cd client/
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
cd client/
npm run lint
```
